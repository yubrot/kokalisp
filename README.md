# kokalisp

kokalisp is a [Koka](https://github.com/koka-lang/koka) version of [Rosetta Lisp](https://github.com/yubrot/rosetta-lisp) implementation.

```bash
# Run Rosetta Lisp tests
$ make test

# Build an executable
$ make

# Run REPL
$ ./kklisp

# Run some example
$ ./kklisp rosetta-lisp/examples/conways-gol.lisp
```
